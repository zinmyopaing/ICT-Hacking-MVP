# ICT-Hacking-MVP

## Selected IT Venture Direction

Our team selected a Campus Lost and Found System.

The target users are university students who lose personal items on campus. Currently, students rely on chat groups, friends, and security staff to find lost items. Our proposed solution is a web-based platform where students can report and search for lost items easily.
