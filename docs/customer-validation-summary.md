## 1. Project Title

**Smart Study Group Finder**

---

## 2. Prototype Tested

The Smart Study Group Finder MVP was tested with university students to evaluate whether users could easily find study groups, create new study groups, join existing groups, and receive notifications without assistance.

- **Prototype version:** V1.0
- **Prototype link or screenshot location:** /prototype/ (or GitHub screenshots folder)
- **Main tasks tested:**
  - Register a new account
  - Login to the system
  - Find study groups
  - View group details
  - Create a study group
  - Join a study group
  - View notifications
- **Related requirements from `system-requirements.md`:**

  - FR-03 User Login

## 4. Validation Method

The prototype was tested individually by students. Each participant completed the assigned tasks while the team observed user behaviour, recorded completion time, noted confusion points, and collected feedback using the validation form.

- **Testing method:** Moderated usability testing
- **Date/time:** July 2026
- **Location or online platform:** Rangsit University Computer Laboratory
- **Number of testers:** 12 students
- **Data collected:**
  - Task completion
  - Completion time
  - User feedback score
  - Interest level
  - Confusion points
  - Improvement suggestions

---

## 5. Summary of Results

| Metric | Result | Interpretation |
|---------|--------|----------------|
| Total test users | 12 | Sufficient sample for MVP validation |
| Task success rate | 91.7% | Most users completed the assigned tasks successfully |
| Average feedback score | 4.5 / 5 | Users were satisfied with the prototype |
| Average interest level | 4.7 / 5 | Students showed strong interest in using the system |
| Most common confusion point | Search by Course Code | Search functionality should be improved |

---

## 6. Key User Comments

- The Find Groups page was simple and easy to use.
- The Join Group feature was useful but users wanted clearer instructions.
- Some users found the course code search slightly confusing.
- Students liked the Create Group feature for organizing study sessions.
- Notifications should be easier to read and highlight important updates.

---

## 7. Affected Requirements

| Requirement ID | Evidence Found | Required Prototype Improvement |
|----------------|----------------|--------------------------------|
| FR-04 | Users experienced difficulty searching by course code | Improve search filters and course code search |
| FR-06 | Some users felt the Create Group form contained unnecessary fields | Simplify the form |
| FR-07 | Users hesitated before joining a group | Improve Join Group instructions and button visibility |
| FR-08 | Users requested clearer notification messages | Improve notification design and message clarity |

---

## 8. Conclusion

The Smart Study Group Finder MVP was **validated with minor revisions**.

Most participants successfully completed the required tasks and expressed positive feedback about the system. The experiment confirmed that the project direction is appropriate. However, improvements to the search functionality, Create Group form, Join Group workflow, and Notifications should be completed before developing the final prototype.
