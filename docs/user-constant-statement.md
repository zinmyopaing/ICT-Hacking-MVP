# User Consent Statement

This prototype was developed for the ICT105 course project, **Smart Study Group Finder**.

By participating in this prototype evaluation, you voluntarily agree to test the prototype and provide feedback. The information collected will be used only for academic purposes, usability evaluation, prototype improvement, and final project reporting.

Participation is voluntary. You may stop participating at any time. No unnecessary personal or sensitive information will be collected. All responses will be kept confidential and used only by the project team.

---

## Data Collected

- Anonymous Test User ID (e.g., TU-01)
- User Role (Student)
- Task Completion Status
- Task Completion Time
- Feedback Score (1–5)
- Interest Level (1–5)
- Usability Comments
- Confusion Points
- Most Useful Feature

---

## Data Use

The collected information will only be used for:

- Customer validation analysis
- Prototype usability evaluation
- Improving the Smart Study Group Finder prototype
- ICT105 coursework and final project documentation

No personal information will be shared with third parties or used for commercial purposes.
