# Feature-Value Mapping

| Feature ID | MVP Feature | Requirement ID | User Story ID | User Value | Business / Operational Value | Prototype Screen or Module | Priority |
|------------|-------------|----------------|---------------|------------|------------------------------|----------------------------|----------|
| F-01 | Search Study Groups | FR-01 | US-01 | Students can quickly find study groups for their subject. | Helps students find suitable groups and increases platform usage. | search-group.html | Must |
| F-02 | Create Study Group | FR-02 | US-02 | Students can create a new study group easily. | Increases the number of study groups available. | create-group.html | Must |
| F-03 | Join Study Group | FR-03 | US-03 | Students can join a study group with one click. | Makes group management easier and increases active users. | join-group.html | Must |
| F-04 | View Group Details | FR-04 | US-04 | Students can view group information before joining. | Helps students choose the right study group. | group-details.html | Should |

## Reflection

**Which feature creates the strongest user value?**

Search Study Groups (F-01) because students can quickly find the best study group for their needs.

**Which feature creates the strongest business or operational value?**

Join Study Group (F-03) because it increases user activity and keeps study groups active.

**Which required feature is still weak or unclear?**

View Group Details (F-04) because the exact information to display may need more planning.
