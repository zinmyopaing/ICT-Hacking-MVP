# Lab 08 - Analytics Insights

## 1. Analytics Question

What did students do, say, and prove while testing the Smart Study Group Finder MVP?

---

## 2. Metrics Calculated

| Metric | Formula / Method | Result |
|---------|------------------|--------|
| Total test users | Count testers | 12 |
| Completed tasks | Count Task_Completed = Yes | 11 |
| Task success rate | Completed tasks / Total testers | 91.7% |
| Average feedback score | Average of 1–5 ratings | 4.5 / 5 |
| Average interest level | Average of 1–5 ratings | 4.7 / 5 |
| Confusion points | Count repeated confusion categories | 4 |

---

## 3. Findings

### 1.
Most students successfully completed the main tasks such as finding a study group, joining a group, and viewing notifications without assistance.

### 2.
Some students experienced difficulty when searching by course code and understanding the Join Group process.

### 3.
Students rated the Find Groups feature as the most useful function and expressed strong interest in using the system for future study collaboration.

---

## 4. Interpretation

The validation results indicate that the Smart Study Group Finder MVP successfully addresses the main problem of helping students discover and join study groups.

The overall task completion rate and user satisfaction are high. However, improvements are needed for the search filters, course code search, and Join Group workflow before moving to the final implementation stage.

---

## 5. Requirements Affected

## 6. Next Prototype Improvement

- Improve the Find Groups search filter.
- Simplify the Create Group form.
- Make the Join Group process easier to understand.
- Improve course code search accuracy.
- Redesign the Notifications page for better readability.
- Add clearer success and confirmation messages after creating or joining a group.
