---

## 2. Evidence Supporting the Decision

The customer validation results show that the Smart Study Group Finder MVP successfully solves the main problem of helping students find and join study groups.

Evidence collected during testing includes:

- 12 university students participated in the validation test.
- 11 out of 12 students successfully completed all assigned tasks.
- Task success rate: **91.7%**
- Average feedback score: **4.5 / 5**
- Average interest level: **4.7 / 5**
- Most users found the Find Groups feature easy to use.
- Some users experienced minor confusion when searching by course code and using the Join Group feature.

These results indicate that the project direction is correct and only minor improvements are required before final implementation.

## 4. Requirements to Improve

| Requirement ID | Problem Found | Improvement Needed |
|----------------|---------------|--------------------|
| FR-04 | Some students had difficulty searching by course code. | Improve search filters and course code search. |
| FR-06 | The Create Group form contains unnecessary fields. | Simplify the form and improve layout. |
| FR-07 | Some users were unsure before joining a group. | Improve Join Group instructions and button visibility. |
| FR-08 | Notification messages were not obvious enough. | Improve notification design and highlight unread updates. |

---

## 5. Prototype Changes Before Next Lab

The team will make the following improvements before developing the final prototype:

- Improve the Find Groups search filter.
- Add better support for course code search.
- Simplify the Create Group form.
- Improve the Join Group workflow.
- Redesign the Notifications page.
- Add clearer confirmation messages after users create or join a study group.
- Improve overall user interface consistency across all pages.
