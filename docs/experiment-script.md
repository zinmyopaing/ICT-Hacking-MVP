# Experiment Script

## 1. Opening Script

Thank you for helping us test our Smart Study Group Finder prototype.

This is **not a test of your skills**. We are testing whether our website is easy to understand and use.

Please use the prototype naturally and think aloud while completing each task. If anything is confusing, difficult, or unclear, please tell us. Your honest feedback will help us improve the final version of our project.

---

## 2. Test Scenario

Imagine that you are a university student who wants to prepare for an upcoming exam. You need to find classmates who are studying the same subject, join an existing study group, or create a new study group if you cannot find one.

Use the Smart Study Group Finder prototype to complete the following tasks.

---

## 3. Tasks for Tester

| Task No. | Task Instruction | Related Requirement | Expected Result |
|----------|------------------|---------------------|-----------------|
| T1 | Open the Homepage and explain what this website is for. | FR-01 | The user understands the purpose of the website. |
| T2 | Register a new account or log in to the system. | FR-02, FR-03 | The user successfully accesses the system. |
| T3 | Search for a Programming Study Group using the Find Groups page. | FR-04 | The user finds a suitable study group. |
| T4 | Open the Group Details page and review the group information. | FR-05 | The user understands the group's schedule, members, and description. |
| T5 | Submit a Join Group request. | FR-07 | The user successfully submits a join request. |
| T6 | Create a new study group by completing the Create Group form. | FR-06 | The system accepts the new study group information. |
| T7 | Open the Notification page and review recent notifications. | FR-08 | The user understands recent updates and notifications. |

---

## 4. Observation Notes

During the experiment, observe and record:

- Where the user hesitates.
- Which buttons or menus are difficult to find.
- Whether the user completes each task without assistance.
- Any mistakes or unexpected actions.
- Comments or suggestions provided by the tester.
- Time required to complete each task.
